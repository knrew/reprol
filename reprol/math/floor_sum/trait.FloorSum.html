Expand description

floor sumを計算するトレイト．

Required Methods§

Source

fn checked_floor_sum(n: Self, m: Self, a: Self, b: Self) -> Option<Self>

sum_{i=0}^{n-1} floor((a*i + b) / m) を返す．

オーバーフローまたは不正な引数の場合はNoneを返す． u128/usize/isizeでは，i128に変換できない入力もNoneとなる．

Provided Methods§

Source

fn floor_sum(n: Self, m: Self, a: Self, b: Self) -> Self

sum_{i=0}^{n-1} floor((a*i + b) / m) を返す．

§Panics
  • m == 0 の場合
  • 符号付き整数型で n < 0 の場合
  • 結果がオーバーフローする場合
  • u128/usize/isizeで入力がi128に変換できない場合

Dyn Compatibility§

This trait is not dyn compatible.

In older versions of Rust, dyn compatibility was called "object safety", so this trait is not object safe.
